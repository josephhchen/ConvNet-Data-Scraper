# ConvNet Data Scraper

## A Modified Data Scraper with Convolutional Neural Network (CNN) Classification for Object Detection in Videos

This project is a tool that integrates web scraping, frame extraction, and image classification using a convolutional neural network (CNN). Developed for ML research at The University at Buffalo, this tool is designed to scrape video data from online sources, process the frames, and identify content that contains a specified object using advanced CNN techniques. Once identified, the tool compiles these frames into a new video for further analysis or research purposes.
